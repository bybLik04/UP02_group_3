Линейная регрессия - один из основных методов статистического моделирования, используемый для изучения отношений между зависимой переменной и одной или несколькими независимыми переменными. В линейной регрессии предполагается, что зависимость между переменными является линейной.

Модель линейной регрессии имеет следующий вид:
y = β₀ + β₁x₁ + β₂x₂ + ... + βₚxₚ + ε

Где:
- y - зависимая переменная, которую мы пытаемся предсказать.
- x₁, x₂, ..., xₚ - независимые переменные.
- β₀, β₁, β₂, ..., βₚ - коэффициенты, которые определяют вклад каждой независимой переменной в предсказание зависимой переменной.
- ε - случайная ошибка.

Оценка коэффициентов модели линейной регрессии также производится с использованием метода наименьших квадратов, минимизирующего сумму квадратов разностей между фактическими значениями зависимой переменной и предсказанными значениями, полученными с помощью модели.

Примеры использования линейной регрессии:
1. Прогнозирование цен на недвижимость: Используя данные о продажах недвижимости, размерах и расположении домов, можно построить модель линейной регрессии для прогнозирования цен на недвижимость и определения влияния каждого фактора.

2. Маркетинговые исследования: Линейная регрессия может быть применена для изучения влияния маркетинговых затрат на объем продаж или оценки эффективности маркетинговых кампаний.

3. Медицинские исследования: Линейная регрессия может использоваться для анализа медицинских данных, например, для изучения влияния факторов, таких как возраст, пол и генетические показатели, на развитие определенных заболеваний.
